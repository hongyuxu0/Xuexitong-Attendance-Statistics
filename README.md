# 学习通签到统计工具

基于Python+tkinter开发的图形化工具，用于批量处理学习通导出的签到数据（CSV/Excel格式），自动统计学生签到状态并生成汇总表。


## 功能特点
1. **多方式导入**：支持单个文件、批量文件、整个文件夹的签到数据导入；
2. **智能识别表头**：自动检测包含“签到状态”的表头行，适配学习通不同导出格式；
3. **单文件统计**：为每个导入的签到文件生成带“签到统计”列（1=已签、0=未参与）的结果文件；
4. **多文件汇总**：统计所有导入文件中学生的总签到次数，按班级/专业分类展示；
5. **操作日志**：实时记录导入、处理、保存等操作，便于排查问题。


## 使用步骤
1. **运行工具**：
   - 直接双击打包后的`学习通签到统计工具.exe`（若有）；
   - 或本地运行代码：`python 学习通签到统计APP.py`。
2. **导入数据**：选择“导入单个文件/批量文件/导入文件夹”，选择学习通导出的签到CSV/Excel文件；
3. **处理文件**：点击“处理选中文件”，工具自动生成单文件统计结果；
4. **生成汇总表**：点击“生成汇总表”，选择保存路径，获取所有学生的总签到次数汇总表。
